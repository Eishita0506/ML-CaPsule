# Rick Sanchez Chatbot using DialoGPT

## Project Overview

This project builds an AI-powered chatbot inspired by Rick Sanchez from *Rick and Morty* using Microsoft DialoGPT and the Hugging Face Transformers library.

The chatbot is fine-tuned on Rick and Morty dialogue data so that it can generate conversational responses in a Rick-like style. A simple Gradio interface is also included to test the chatbot in real time.

## Features

- Preprocesses Rick and Morty dialogue data
- Fine-tunes Microsoft DialoGPT on character-style conversations
- Generates chatbot responses from user input
- Provides an optional Gradio web app for real-time testing
- Easy setup and reproducible workflow
